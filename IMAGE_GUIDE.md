# 이미지 가이드

## 필요한 이미지 파일

### 1. 픽셀아트 이미지 (필수)
- **경로**: `/public/pixel-art.gif`
- **용도**: "우리의 이야기" 섹션에 표시되는 메인 이미지
- **형식**: GIF (애니메이션 가능) 또는 PNG/JPG
- **권장 크기**: 가로 400-600px, 세로 비율 자유
- **설명**: 커플의 픽셀아트 스타일 이미지 1장

### 2. 갤러리 이미지 (선택, 최대 3장)
- **경로**: 
  - `/public/gallery-1.jpg`
  - `/public/gallery-2.jpg`
  - `/public/gallery-3.jpg`
- **용도**: 갤러리 섹션에서 좌우 스와이프로 볼 수 있는 사진들
- **형식**: JPG, PNG
- **권장 크기**: 가로 800-1200px, 세로 비율 4:3 또는 16:9
- **설명**: 결혼 사진, 데이트 사진 등 커플 사진 1-3장

## 이미지 추가 방법

1. `public` 폴더에 이미지 파일을 복사합니다
2. 파일명을 정확히 맞춰주세요:
   - 픽셀아트: `pixel-art.gif` (또는 `.png`, `.jpg`)
   - 갤러리: `gallery-1.jpg`, `gallery-2.jpg`, `gallery-3.jpg`
3. 브라우저를 새로고침하면 이미지가 표시됩니다

## 참고사항

- 이미지가 없으면 자동으로 플레이스홀더가 표시됩니다
- 갤러리 이미지는 좌우 스와이프(모바일) 또는 드래그(PC)로 전환할 수 있습니다
- 이미지 파일 크기는 최적화를 권장합니다 (각 500KB 이하)
